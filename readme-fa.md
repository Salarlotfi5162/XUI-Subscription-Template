# ⚡️ خوش‌آمدید به قالب پرمیوم اشتراک XUI Panel ⚡️

> **اگر به توضیحات انگلیسی نیاز دارید، فایل [Readme for English](readme.md) را ببینید.**

> **این یک قالب اشتراک سفارشی، تاریک (Dark Mode) و بسیار زیبا مخصوص پنل‌های ثنایی است که امکانات ویژه‌ای مثل کانفیگ نمایشی حجم، لیست جدای لینک‌ها و تب برنامه‌ها را ارائه می‌دهد.**  

## نصب و بروزرسانی  

از آنجا که این یک فورک اختصاصی است، روش نصب به این صورت می‌باشد:

```bash
cd /opt
git clone git@github.com:Salarlotfi5162/XUI-Subscription-Template.git DVHOST
cd DVHOST
npm install
```  

## فایل پیکربندی  

```bash
nano /opt/DVHOST/dvhost.config
```  
پس از تغییر فایل پیکربندی، باید سرویس را مجدداً راه‌اندازی کنید.  

```bash
systemctl restart DVHOST_TEMPLATE
systemctl status DVHOST_TEMPLATE
```  
**شما نباید از پورت‌های 2082 و 2083 استفاده کنید.**  

## ویژگی‌های جدید قالب پرمیوم
- **ظاهر بی‌نظیر تاریک:** طراحی Dark Mode پیشرفته با جزئیات رنگی مدرن.
- **محاسبه هوشمند حجم:** برای پنل‌های ثنایی که حجم را روی هر اینباند مجزا محاسبه می‌کنند، این قالب حجم کل را بر تعداد اینباندهای کاربر تقسیم می‌کند تا حجم واقعی را به دست بیاورد.
- **کانفیگ نمایشی (Dummy Config):** یک کانفیگ VLESS اطلاعاتی حاوی متن (مصرف کاربر و روزهای باقیمانده) می‌سازد و به طور مستقیم بالای لیست کانفیگ‌ها در نرم‌افزارهای VPN به کاربر نشان می‌دهد.
- **سیستم تب‌بندی پویا:** نمایش لیست کامل اپلیکیشن‌های پیشنهادی بر اساس سیستم‌عامل، صفحه‌ی اختصاصی تولید QR لینک، و صفحه‌ی لیست کانفیگ‌ها مجزا.

از این قالب بی‌نقص و زیبا برای سرویس‌های خود لذت ببرید!
